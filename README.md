# Công tác sinh viên
Đây là ứng dụng cho phép sinh viên theo dõi các thông báo của phòng công tác sinh viên.

## Chức Năng

### Đăng nhập/ Đăng ký

- Đăng nhập
- Đăng ký
- Quên mật khẩu

### Trang Chủ

- Trang chủ
- Search thông báo theo tiêu đề

- Lưu các thông báo quan trọng vào xem sau
